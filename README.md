# The Devil’s Casino
**The Devil’s Casino** es una app de casino en desarrollo hecha con **Android (Jetpack Compose)** y **Spring Boot**.

---

## Estado del proyecto
**Versión actual:** `v0.5.0`  
Proyecto en desarrollo.

---

## Qué tiene ahora
* Login y creación de usuarios funcional.
* Navegación entre pantallas.
* **Lobby** con selección de juegos.
* **Blackjack funcional** con apuestas en Devil Coins (DC).
* Wallet persistente y conectada a base de datos.
* **Settings**:
  - Cambio de idioma.
  - Selección de música de fondo.
* Música de fondo reproducible en la app.
* **Oferta diaria** en Wallet: 100 Devil Coins cada 24 horas.
* Backend con API REST y base de datos MySQL.

---

## Juegos

* **Blackjack** → funcional con guardado de partidas.
* **Dados Mentirosos** → solo interfaz (sin lógica aún).
